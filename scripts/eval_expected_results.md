# Search Eval Expected Results

이 문서는 `scripts/eval_questions.txt`의 질문과 1:1로 대응하는 최소 평가 기준이다.

목적:
- 검색 튜닝 전/후 결과를 같은 기준으로 비교하기
- 질문별로 "무엇이 1위로 와야 하는가"를 명확히 정하기
- 데모/발표 시 검색 품질 검증 기준으로 사용하기

주의:
- `expected_top_1`은 현재 프로젝트에서 우선 기대하는 대표 근거다.
- 일부 항목은 현재 원문 법령보다 `seed` 설명이 1위가 더 적절하도록 설계되어 있다.
- `seed` 항목은 이후 원문 법령/가이드라인으로 대체될 수 있다.
