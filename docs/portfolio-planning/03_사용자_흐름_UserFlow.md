# 사용자 흐름 (User Flow)

## 메인 플로우
1. 홈 진입
2. 오늘의 노동체크 선택
3. 상담 시작하기 클릭
4. AI 상담실에서 질문 입력
5. (선택) 증거 파일 첨부 후 전송
6. 근거 포함 답변 수신
7. 상담내역에서 세션 선택
8. 체크리스트(먼저 할 일/준비할 서류) 확인 및 체크 저장
9. 알림 누적 내역 확인
10. 증거보관함에서 파일 추가 관리

## 세부 플로우
- 첨부 파일 선택 플로우
  - 첨부 버튼 클릭 → 검색/카테고리 필터 → 파일 다중 선택 → 전송
- 체크리스트 저장 플로우
  - 상담내역 선택 → 항목 체크/해제 → 서버 저장 → 재진입 시 복원
- 알림 조회 플로우
  - 상담 선택 → 해당 `chat_session_id` 필터로 자동 알림 조회
