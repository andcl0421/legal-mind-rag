# 화면 설계서 (Wireframe + 정책)

## 1. Home
### 구성
- 상단 브랜드/아이콘
- 메인 배너 + 상담 시작 버튼
- 오늘의 노동체크 카드
- 노무톡톡과 대화하기 카드

### 상태 변화
- 감정 선택 시 활성 스타일 변경
- 감정별 맞춤 문구 노출

### 예외 케이스
- 미로그인 상태에서도 홈 탐색 가능

---

## 2. Chat
### 구성
- 세션 목록(좌)
- 채팅 영역(중앙)
- 참고문서 뷰어(우)
- 입력창 하단 첨부 패널

### 상태 변화
- 메시지 전송 중 로딩 상태
- 첨부 패널 열기/닫기
- 첨부 검색/카테고리 필터 실시간 반영
- 선택 첨부 칩 표시/해제

### 예외 케이스
- 토큰 없음: 전송 차단 및 안내
- 세션 삭제 후 현재 세션 재선정

---

## 3. History
### 구성
- 탭 필터 + 검색 + 상담 테이블
- 우측 체크리스트/알림 패널

### 상태 변화
- 상담 row 선택 시 우측 패널 갱신
- 체크박스 변경 시 즉시 저장

### 예외 케이스
- 상담 미선택 시 안내 문구
- 해당 세션 알림/체크리스트 없음 처리

---

## 4. MyPage / Evidence Vault
### 구성
- 파일 업로드 폼
- 카테고리/세션 연결
- 파일 목록(미리보기/다운로드/삭제)

### 상태 변화
- 업로드 중 버튼 상태
- 필터별 목록 갱신

### 예외 케이스
- 파일 없음/용량 초과/형식 오류 안내
