# 서비스 개요서 (1p)

## 프로젝트명
- 노무톡톡 (NomuTalkTalk)

## 문제 정의
- 일반 근로자는 노동 이슈(임금, 해고, 휴가, 계약 등) 발생 시 법령/절차를 빠르게 파악하기 어렵다.
- 상담 후에도 “지금 무엇을 해야 하는지”, “어떤 서류를 준비해야 하는지”가 구조화되지 않아 실행으로 이어지지 않는 문제가 있다.

## 타겟 사용자
- 노동법 전문지식이 없는 일반 근로자
- 이직/퇴사/육아휴직/해고 등 민감 상황에서 빠른 가이드가 필요한 사용자

## 해결 방식
- RAG + LLM 기반 상담으로 질문에 대한 근거 중심 답변 제공
- 상담 결과를 `먼저 할 일` / `준비할 서류`로 구조화
- 상담 세션 단위로 알림, 체크리스트, 증거 파일을 누적 관리

## 핵심 가치
- 이해하기 쉬운 상담
- 실행 가능한 결과물
- 기록/증거/이력의 일관된 관리
