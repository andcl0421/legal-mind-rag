# 🎨 04. 프론트엔드 및 UI/UX 설계 (Frontend Plan)

## 1. 디자인 원칙 (Design Principles)
- **신뢰성 (Trust)**: 법률 서비스인 만큼 차분한 블루/화이트 톤 사용.
- **가독성 (Readability)**: 복잡한 법 조항을 보여줄 때 폰트 크기와 줄 간격을 넉넉히 배치.
- **반응형 (Responsive)**: 모바일에서도 노동 현장에서 즉시 확인할 수 있도록 최적화.

## 2. 주요 화면별 상세 구성

### ① 메인 대시보드 (Main Dashboard)
- **상단**: 닉네임과 함께 "OOO 님은 현재 [5인 이상] 사업장 기준을 적용받고 있습니다" 문구 노출.
- **중앙**: 최신 노무 이슈(카드 뉴스 형태) 슬라이드.
- **하단**: [AI 상담 시작], [급여 계산기], [나의 상담 리포트] 퀵 버튼.

### ② AI 상담실 (Chat Interface) - 핵심 화면
- **왼쪽(메인)**: 카카오톡 스타일의 채팅창. 
  - AI 답변 시 출처(법령 조항)가 클릭 가능한 링크나 칩(Chip) 형태로 표시됨.
- **오른쪽(사이드바/탭)**: 
  - **참조 문서 뷰어**: AI가 참고한 PDF의 해당 페이지를 미리보기로 보여줌.
  - **계산기 위젯**: 상담 도중 연차나 수당 계산이 필요할 때 즉시 호출.

### ③ 마이페이지 및 리포트 (History & Profile)
- 과거 상담 내역을 날짜별/키워드별로 검색 가능.
- 특정 상담 건을 **'PDF 리포트 다운로드'** 하는 기능.
- 내 정보 수정: 업종, 근로자 수 등을 변경하면 AI 상담 로직에 즉시 반영.

## 3. 사용자 여정 (User Journey Map)
1. **진입**: 소셜 로그인 또는 자체 로그인.
2. **정보 입력**: 첫 방문 시 업종/근로자 수 설정 (상담 정확도 향상).
3. **상담**: 궁금한 사항 질문 -> AI가 법령 근거와 함께 답변.
4. **확인**: 답변 내용 중 어려운 용어 클릭 시 툴팁(Tooltip) 설명 제공.
5. **저장**: 상담 결과 저장 및 나중에 보기.

## 4. 기술 스택 (Frontend Tech Stack)
- **Framework**: React.js (또는 Next.js)
- **Styling**: Tailwind CSS (빠르고 깔끔한 디자인)
- **State Management**: Zustand (가볍고 직관적인 상태 관리)
- **Icons**: Lucide React (심플한 아이콘 세트)

## 5. 고급화를 위한 특화 기능 (Premium Features)
- **AI-to-Human 브리핑**: AI 상담 이력을 요약하여 전문 노무사 상담 시 전달하는 브리핑 파일 생성.
- **인터랙티브 법령 뷰어**: 답변 내 법률 근거 클릭 시 실제 법령 PDF의 해당 섹션 하이라이트 노출.
- **증거 보관함 (Security Vault)**: 근로계약서, 현장 사진 등을 보안 저장하여 사후 증거력 확보.
- **지능형 진정서 초안 작성**: 상담 데이터를 기반으로 노동청 제출용 진정서 표준 양식 자동 렌더링.
